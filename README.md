## 🚀 精选项目

### Dynamic Shader

> 基于 HLSL 开发的自定义 Shader，使游戏《Stardew Valley》实现 HD2D 效果。

![Downloads](https://img.shields.io/badge/Downloads-51,600+-brightgreen?style=for-the-badge)

- 利用 Harmony 拦截游戏渲染层，注入自定义 Shader 模拟 3D 光照系统
- GPU 加速 + 双缓冲阴影收集队列，实现低开销的全局阴影渲染
- 自定义顶点/像素着色器：伪 3D 投影、接触硬化阴影、环境光色相偏移
- Dual Kawase + 降采样大幅加速模糊处理，双 Dict 纹理分类减少约 70% draw call

**技术栈:** &nbsp; `Shader` &nbsp; `HLSL` &nbsp; `GPU Batching` &nbsp; `Harmony`

### BetterBuildingUpgrades

> 使用 Harmony 和 SMAPI 框架扩展游戏《Stardew Valley》核心方法的游戏模组。

![Downloads](https://img.shields.io/badge/Downloads-12,700+-brightgreen?style=for-the-badge)
![Endorsements](https://img.shields.io/badge/Endorsements-190+-blue?style=for-the-badge)

- 通过反射注入代码，重写并扩展游戏核心方法
- 解决多人模式下的数据一致性问题
- 优化大范围自动化逻辑的计算开销，确保游戏帧率稳定

**技术栈:** &nbsp; `C#` &nbsp; `SMAPI` &nbsp; `Harmony`
